# Kafka Message Consumer

## Опис проекту
Простий Kafka-споживач для отримання та обробки повідомлень з топіка з гнучкими налаштуваннями.

## Вимоги
- Python 3.8+
- confluent-kafka
- typing

## Встановлення залежностей
```bash
pip install confluent-kafka
```

## Налаштування

### Змінні середовища
- `BOOTSTRAP_SERVERS`: Адреса Kafka брокера (за замовчуванням `localhost:9092`)
- `TOPIC`: Назва топіка для споживання повідомлень
- `GROUP_ID`: Ідентифікатор групи споживачів

## Використання

### Базовий запуск
```python
from kafka_consumer import SimpleKafkaConsumer

consumer = SimpleKafkaConsumer(
    bootstrap_servers='localhost:9092',
    topic='ВашТопік',
    group_id='ваша_група_споживачів'
)

# Отримання максимум 10 повідомлень
consumer.consume(max_messages=10)
```

### Параметри налаштування
- `bootstrap_servers`: Адреса Kafka брокера
- `topic`: Назва топіка
- `group_id`: Ідентифікатор групи споживачів
- `timeout`: Час очікування повідомлень
- `max_messages`: Максимальна кількість повідомлень

## Функціонал
- Підключення до існуючого топіка
- Парсинг JSON-повідомлень
- Гнучке налаштування споживання
- Обробка помилок Kafka

## Обробка помилок
- Автоматична обробка помилок підключення
